PAWS Finder

Source: http://www.youtube.com/watch?v=YE7VzlLtp-4

PAWS Finder is a full stack web application that enables users to find adoptable dogs and shelter information.

  • Backend: Python, PostgreSQL, SQLAlchemy, Flask
  • Frontend: Jinja, Bootstrap, HTML5, CSS3, Javascript (jQuery, AJAX)
  • APIs: Petfinder, Google Maps, Zipcode
